HTML Fundamentals

HTML provides structure — the skeleton of the page. Key concepts from the PDF: elements, tags, attributes, semantic elements, lists, images, links and forms. (See pages 2–14 in the PDF.) fileciteturn1file0

Basic structure

<!doctype html>
<html lang="en">
  <head><!-- meta, title, links --></head>
  <body><!-- visible content --></body>
</html>

Common tags

  • <h1>…</h1> headings
  • <p> paragraph
  • <a href=""> links (use target="_blank" rel="noopener" for external)
  • <img src="" alt=""> images (provide alt text)
  • <ul>/<ol> <li> lists
  • <form> inputs and controls

CSS Fundamentals

CSS controls presentation — colors, spacing, layout, and responsive rules. Concepts: selectors, properties, the box model, flexbox, and media queries (see pages 16–23). fileciteturn1file0

Selectors

Element selector (p), class selector (.btn), id selector (#header), universal (*).

Box model

Content → padding → border → margin. Use box-sizing: border-box to make sizing easier.

JavaScript Fundamentals

JavaScript adds interactivity. Key concepts: variables (let, const), types, functions, the DOM, events and control flow (if/else, loops). (See pages 26–34.) fileciteturn1file0

DOM & Events

Use document.querySelector and addEventListener to react to user input.
